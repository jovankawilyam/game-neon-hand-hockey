\# 🎮 Game Neon Hand Hockey



Game interaktif berbasis webcam yang menggunakan hand tracking (MediaPipe) untuk mengontrol paddle dengan jari.



\## 🚀 Demo

https://muhammadreki.github.io/Game-Neon-Hand-Hockey/



\## 🧠 Fitur

\- ✋ Kontrol pakai jari (index finger tracking)

\- 🎮 Multiplayer 2 player (1 layar)

\- ⚡ Real-time tracking

\- 🌈 Neon visual effect



\## 🛠️ Teknologi

\- HTML5 Canvas

\- JavaScript

\- MediaPipe Hands



\## 🎯 Cara Main

\- Player 1: sisi kiri (tangan kiri)

\- Player 2: sisi kanan (tangan kanan)

\- Gerakkan jari untuk kontrol paddle
